# CMS Realtime Scoring by Rank

This tool let you to use King of the Hill rule in CMS.

## Requirements
* Ubuntu 16.04
### python packages
* certifi==2019.3.9
* chardet==3.0.4
* future==0.17.1
* gevent==1.4.0
* greenlet==0.4.15
* idna==2.8
* requests==2.21.0
* six==1.12.0
* urllib3==1.24.1

## Run
1. Copy config.cfg.sample to config.cfg and edit it.
2. `pip install -r requirements.txt`
3. `python score.py`
